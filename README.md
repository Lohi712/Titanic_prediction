# Titanic_prediction
This project builds a machine learning model to predict whether a passenger on the Titanic survived or not with an accuracy of ~84%.

The project includes:

Data cleaning & preprocessing
Exploratory Data Analysis (EDA)
Feature engineering
Model training using Random Forest
Evaluation using accuracy, classification report, and confusion matrix
Saving the trained model (titanic_model.pkl)

Dataset used: 'https://raw.githubusercontent.com/datasciencedojo/datasets/master/titanic.csv'

Technologies Used:
Python,
Pandas,
NumPy,
seaborn & matplotlib,
Scikit-learn,
Joblib
